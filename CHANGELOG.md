# Changelog

## 0.1.0 (2026-03-17)

### Features
- Changelist management (create, rename, delete, set active)
- Automatic file tracking with configurable polling
- Tree view with drag-and-drop in Source Control sidebar
- **Hunk-level tracking** — split changes in the same file across changelists
- Independent commit per changelist (file-level and hunk-level)
- Colored gutter decorations per changelist
- Inline diff "Move to Changelist" action
- Jira Cloud ticket integration (REST API v3)
- GitHub Issues ticket integration
- Status bar showing active changelist
- Persistent state in `.vscode/changelists.json`
